# Blockly Penguin

[Blockly](https://developers.google.com/blockly/) Customize sample


## Getting started

Install node

ref
- [node.js](https://nodejs.org/ja/)
- [nodebrew](https://github.com/hokaccha/nodebrew)


Install yarn & dependencies

```
$ npm install yarn -g
$ yarn
```

Run http-server
```
$ npm run http-server
```
